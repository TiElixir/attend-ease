# **App Name**: AttendEase

## Core Features:

- Authentication: Secure login using college-provided Google IDs (YYYYXYZ123.firstname@students.iiests.ac.in).
- Schedule Display: Display a personalized class schedule based on the user's college ID and the provided JSON schedule data.
- Attendance Marking: Allow users to mark their attendance status (present, absent, cancelled) for each class.
- Attendance Calculation: Automatically calculate and display the user's attendance percentage for each course and overall.
- Data Persistence: Store attendance data persistently using Firestore, so user attendance data is retained across sessions.

## Style Guidelines:

- Primary color: #576CBC (a cool blue), evoking trust and reliability.
- Background color: #F0F4F8 (very light blue), providing a clean and unobtrusive backdrop.
- Accent color: #A2A8D3 (desaturated violet), used sparingly for interactive elements to add a touch of sophistication.
- Body and headline font: 'Inter', a sans-serif for clean and modern readability.
